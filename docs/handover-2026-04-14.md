## 미커밋 변경사항 (이번 세션 종료 시점)

아래 명령어로 한 번에 커밋 가능:

```bash
cd "C:/Users/User/kendo-video-app"

git add app/admin/page.tsx "app/video/[id]/page.tsx" components/BottomNav.tsx components/Sidebar.tsx components/VideoForm.tsx app/list/ fetch_videos.py fix_participants.py videos_template_fetched.csv

git commit -m "feat: 영상 삭제 기능, 리스트 뷰, 자동완성 가이드, 모바일 수정 등 다수 개선"

git push origin main
```

> ⚠️ `client_secret.json`은 Google OAuth 인증키 — 절대 커밋 금지. `.gitignore`에 추가 권장.

## 알려진 이슈 / 주의사항

| 항목 | 내용 |
|------|------|
| `/participant` 라우트 | nav에서 제거됐지만 파일(`app/participant/page.tsx`)은 남아있음 — 필요 시 삭제 가능 |
| `client_secret.json` | YouTube OAuth 키, 프로젝트 루트에 있음. gitignore 추가 필요 |
| `videos_template_fetched.csv` | fetch_videos.py 실행 결과물. 커밋해도 무방하나 민감 정보 없음 |
| 참가자 태그 링크 | 영상 상세 페이지 참가자 태그에서 `/participant` 링크 제거됨 (단순 span). 필요 시 `/list?participant=이름` 형태 필터 연동 고려 |
| CSV 업로드 미리보기 | 파일 선택 후 미리보기 표시, 실제 업로드는 별도 버튼 클릭 필요 |
