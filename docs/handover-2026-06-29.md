## 배포 전 필수 작업

### 1. Supabase SQL 실행
Supabase Dashboard > SQL Editor에서 `docs/migration-competition-2026.sql` 전체 실행

### 2. Supabase Storage 버킷 생성
Dashboard > Storage > New Bucket
- 이름: `competition-files`
- Public 버킷으로 설정 (팸플릿 등 공개 다운로드용)

### 3. 환경변수 추가 없음
기존 `NEXT_PUBLIC_SUPABASE_URL`, `NEXT_PUBLIC_SUPABASE_ANON_KEY` 그대로 사용

## 기능 사용 흐름

### 대회 등록 (관리자)
1. `/admin` → **대회 관리** 탭
2. **+ 대회 등록** → 대회명(프리셋 or 직접 입력), 연도, 날짜, 대회장, 결과 요약 입력
3. 등록 후 행 펼치기 → **출전자 CSV 업로드** (헤더: `name,gender,division,dan_kyu,result,notes`)
4. **파일 업로드** → 유형 선택 후 PDF/이미지 업로드

### 영상과 대회 연결
- 영상 단건 등록 시 하단 **대회 연결** 드롭다운에서 해당 대회 선택
- 연결된 영상은 홈/목록에도 표시되고, 대회 상세 > **영상 탭**에서도 모아볼 수 있음

### 대회장 관리
- `/admin` → 대회 관리 → **대회장 관리** 서브탭
- 한 번 등록한 대회장은 여러 대회에서 재사용 가능
